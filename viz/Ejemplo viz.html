Algoritmo de Kruskal

Se utiliza el algoritmo de Kruskal para encontrar un árbol recubridor mínimo en un grafo conexo y ponderado, es decir, se pretende buscar un subconjunto de aristas que incluyan todos los vértices formando un árbol, y se minimice el valor de la suma de todas las aristas del árbol. El algoritmo consiste en los siguientes pasos:

  • Se marca la arista con menor valor. Si hay más de una, se elige una cualesquiera.
  • De las aristas restantes, se marca la que tenga menor valor, si hay varias con el mismo valor, se elige una cualesquiera.
  • Repetir el segundo paso siempre que la arista que se elija no forme un ciclo con las que ya están marcadas.
  • El algoritmo finaliza cuando todos los vértices están conectados por las aristas marcadas.
